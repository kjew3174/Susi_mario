# 게임 실행 방법

## 설치

1. Python 3.8 이상이 설치되어 있어야 합니다.
2. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 실행

### 게임 실행
```bash
python main.py
```

### 디버그 모드로 실행
```bash
python main.py --debug
```

### 맵 에디터 실행
```bash
python editor.py
```

## 조작 방법

### 게임 플레이
- **방향키** 또는 **WASD**: 이동
- **스페이스바** 또는 **W/↑**: 점프
- **S/↓**: 웅크리기
- **ESC**: 일시정지

### 맵 에디터
- **0-8**: 블록 타입 선택
  - 0: 빈공간
  - 1: 일반 블록
  - 2: 아이템 블록
  - 3: 강한 적 (박건률)
  - 4: 약한 적 (오유준)
  - 5: 보스 (청마)
  - 6: 커피 (무적 아이템)
  - 7: 일반 아이템
  - 8: 토관 (논술)
- **마우스 왼쪽 클릭**: 블록 배치
- **마우스 오른쪽 클릭**: 블록 제거
- **마우스 휠**: 카메라 위아래 이동
- **방향키/WASD**: 카메라 이동
- **Ctrl+S**: 맵 저장
- **G**: 그리드 표시/숨김
- **ESC**: 종료

## 이미지 추가

이미지 파일을 `assets/images/` 폴더에 추가하세요:
- `player.png`: 플레이어 이미지
- `player_crouch.png`: 웅크리기 이미지 (선택사항)
- `background.png`: 배경 이미지
- `application_form.png`: 수시 원서 이미지
- `bridge.png`: 한강 철교 이미지 (선택사항)
- 기타 블록, 적, 아이템 이미지들

## 맵 파일

맵 파일은 `data/maps/` 폴더에 JSON 형식으로 저장됩니다.
기본 맵은 `level1.json`입니다.
